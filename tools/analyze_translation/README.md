# 技法自動分類システム
翻訳技法を自動分類

## 概要
### システム自体のI/O
**Input**
CSV(列名は`Location`,`Note`,`Highlight_JP`,`Highlight_EN`,`注釈`)

**Output**
入力したCSVに「`文化的要素の対応訳`,`翻訳技法`,`翻訳技法の選出理由`」を連結したCSV

#### AIのI/O
**AI-Input**
- 原文
- 訳文
- 文化的要素
- 注釈（ある場合）

**AI-Output**
- `文化的要素の対応訳`
- `翻訳技法`
- `翻訳技法の選出理由`

### 処理の詳細
1. CSVを読み込み
2. CSVから１列抽出
3. 抽出した要素を分割し、AIのプロンプトに組込み渡す
4. AIの出力を「`文化的要素の対応訳`,`翻訳技法`,`翻訳技法の選出理由`」の形式で受け取る
5. 1~4を繰り返し、すべての要素の分析が終了後、元のCSVに連結したものをCSVファイルとして出力
