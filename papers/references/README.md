## 字幕の制限について
- [**Díaz Cintas & Remael 2007**](https://www.taylorfrancis.com/books/mono/10.4324/9781315759678/audiovisual-translation-subtitling-jorge-d%C3%ADaz-cintas-aline-remael)
    
    映像字幕翻訳の入門書。「6秒ルール」や「圧縮の必要性」が言及されている。
- **Szarkowska&Gerber-Morón 2018**
    
    字幕スピードが理解度や認知不可にどのような影響を与えるかを調査したもの。20cpsまで対応可能という事が分かった。
- **Shinohara 2011**(博士後期課程) 
    
    日英字幕翻訳の分野で私と同じような枠組み(Pedercen2011の分類を使って,映画字幕の分析)で分析している論文。「映像作品の字幕を研究対象としなかった理由」に使えそう。

- **Tyokura 2017 字幕翻訳における文字数制限の妥当性についての検証～1秒4文字は妥当か？～**
    
    字幕（Jap：4 words/s, Eng: 6 words/s）の基準の妥当性について、実証実験にて立証している。しかし、サンプル数少な目(アンケート8名、視線計測5名)なので根拠が弱い。

- [「**”字幕翻訳は1秒4文字”について**」](https://www.eizou-honyaku.com/category/jimaku/014.html)

    諸説あり。
